# AI Receptionist & Clinic Management System

A comprehensive healthcare management platform built with React, Vite, Tailwind CSS, and Framer Motion. The application provides separate portals for Patients and Hospitals/Receptionists.

## üöÄ Features

### For Patients
- **Dashboard**: Overview of upcoming appointments and health stats.
- **Clinic Search**: Find clinics by name, specialty, or location.
- **Appointment Booking**: Real-time slot selection and booking.
- **Profile Management**: Manage personal details and medical history.
- **Appointment History**: View past and upcoming appointments.

### For Hospitals
- **Dashboard**: Operational overview (clinics, doctors, appointments).
- **Clinic Management**: Add and manage multiple clinic locations.
- **Doctor Management**: Onboard doctors and assign them to clinics.
- **Slot Management**: Generate and manage appointment time slots.
- **Reception Desk**: Handle incoming calls and follow-up scheduling.
